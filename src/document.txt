# âœ… **2. What is a *Signal* in Angular?**

A **signal** is a *reactive variable* introduced in Angular 16+.

Example:

```ts

```

### âœ” Characteristics

* Automatically **reactive**.
* When signal value changes â†’ UI updates instantly.
* No need for change detection.

### âœ” How to read/write?
